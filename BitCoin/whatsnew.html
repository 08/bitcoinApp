In addition there is no warning if a fee (0.01BC) will be added to some transactions.

Known issues

  • No warning if a small fee (0.01BC) is added to some (complex) transactions
  • Local node takes a long time to start (up to few minutes) so be patient
  • If you stop the local node (Console/stop) you can't restart it. The only way to restart, is to completely exit the App (also from the background) and then start it again. Note that it will automatically restart in this case, so if you want to stop it again you will have to repeat the Console/stop command
  • Local node does not accept incoming connections (-nolisten). Also you cant connect to it using HTTP-JSON-RPC from a different machine.
  • Backup with email does not work when connected to a remote node
